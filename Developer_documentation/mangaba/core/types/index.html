mangaba::core

Module types 

Source
Expand description

Core data types shared across all modules.

Includes conﬁguration structs (LLMConfig, AgentConfig, TaskConfig, MemoryConfig, OpenRouterConfig), messaging (Message, Role), LLM responses (LLMResponse, ToolCall, ToolResult, TokenUsage), agent/task state (AgentState, AgentStatus, TaskState, TaskStatus, ReActStep), and output (TaskOutput).

All types implement Serialize + Deserialize for JSON interchange.
