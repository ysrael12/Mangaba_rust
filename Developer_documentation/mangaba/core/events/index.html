Global EventBus for decoupled subscribe/emit communication.

Listeners are registered via EventBus::subscribe() and fire synchronously on every EventBus::emit(). Events carry a EventType, source ID, and arbitrary JSON data. Used internally by all modules (agent, crew, ReAct, tools, LLM, memory, guardrails).
